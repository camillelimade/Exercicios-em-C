# Exercícios em Linguagem C
Este repositório foi criado com a finalidade de armazenar exercícios desenvolvidos na linguagem de programação C, como parte da disciplina de Introdução à Lógica de Programação.

## Objetivo
Praticar e consolidar conceitos fundamentais de lógica de programação, utilizando a linguagem C como ferramenta de aprendizado.

## Conteúdos abordados
- Estruturas básicas em C como condicionais, loops, switch, etc.
- Biblioteca que permite usar comandos de entrada e saída de dados.
- Operadores aritméticos e lógicos.
- Manipulação de variáveis

## Organização
Os exercícios estão organizados de acordo com os temas estudados durante a disciplina, a organização se da por pacotes que armazenam as questões de cada volume de atividades, podendo evoluir conforme o avanço do conteúdo.

## Tecnologias utilizadas
- Linguagem C
- Compilador GCC
- Visual Studio Code

## Observações
Este repositório possui caráter educacional e empiríco e tem como foco o aprendizado progressivo. Os códigos podem passar por melhorias e refatorações ao longo do tempo.
